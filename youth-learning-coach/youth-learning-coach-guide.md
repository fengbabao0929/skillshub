# 青少年学习教练公众号写作 Skill 使用指南

## 📋 简介

这是一个专门为**青少年学习教练**打造的公众号内容生成工具，帮助快速创建针对家长焦虑、学习方法、习惯培养等主题的高质量公众号文章。

## 🎯 适用场景

- 教育工作者、学习教练运营公众号
- 家长教育、学习指导类内容创作
- 青少年学习方法推广
- 家长焦虑缓解与引导

## 📦 安装方法

### 方法1：添加到 Claude Code Skill 配置

将 `youth-learning-coach-skill.json` 合并到你现有的 skill 配置文件中。

### 方法2：直接运行

```bash
node youth-learning-coach-generator.js --list-topics
```

## 🚀 快速开始

### 1. 查看所有热门主题

```bash
/youth-topics
```

输出示例：
```
【学习痛点】⭐⭐⭐⭐⭐
  • 不爱学习 ⭐⭐⭐⭐⭐
    孩子对学习缺乏兴趣和动力
  • 沉迷手机 ⭐⭐⭐⭐⭐
    孩子过度使用手机影响学习
```

### 2. 生成爆款标题

```bash
/youth-titles --topic "不爱学习"
```

输出示例：
```
1. 为什么孩子不爱学习？90%家长都做错了
2. 如果孩子不爱学习，请立刻停止做这件事
3. 不爱学习的真相，大多数家长都想错了
...
```

### 3. 生成文章大纲

```bash
/youth-outline --topic "不爱学习" --type "痛点分析"
```

### 4. 生成完整文章

```bash
/youth-pain --topic "不爱学习"
```

## ✍️ 写作要点

### 1. 文章基调

**痛点分析类**：共情、深入、启发性
**方法指导类**：实用、清晰、可操作
**案例故事类**：故事性、感染力、启发性
**家长焦虑类**：理解、安慰、建设性
**习惯培养类**：鼓励、具体、可持续
**考试辅导类**：实用、鼓励、减压

### 2. 标题要点

- 字数控制在25字以内
- 直击家长痛点或焦虑
- 使用数字增加具体感
- 制造好奇心或紧迫感
- 避免标题党但要有吸引力

### 3. 内容结构

**开头**：1-2句话戳中痛点或焦虑，引发共鸣

**正文**：
- 小标题分割，便于阅读
- 每段不超过3行
- 用案例或故事说明观点
- 提供具体可操作的建议

**结尾**：
- 总结核心观点
- 给出行动建议
- 引导关注和转发

### 4. 语言风格

- 亲切易懂，避免专业术语
- 多用"你""我们"拉近距离
- 用问句引发思考
- 用金句便于传播

## 🔧 高级用法

### 自定义主题

如果没有找到合适的主题，可以直接使用任何主题：

```bash
/youth-article --topic "你的自定义主题" --type "方法指导"
```

### 组合使用

先生成标题选择，再生成大纲，最后成文：

```bash
/youth-titles --topic "不爱学习"
/youth-outline --topic "不爱学习" --type "痛点分析"
/youth-pain --topic "不爱学习"
```

1. **选择热点主题**：使用 `/youth-topics` 查看5星主题
2. **标题优先**：先生成多个标题，选择最优

4. **真实案例**：用真实案例增强可信度
