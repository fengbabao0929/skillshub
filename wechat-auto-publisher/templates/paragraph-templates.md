# 八宝说财审风格段落模板库

## 1. 开头模板

### 1.1 数字冲击型开头

**模板：**
```markdown
> [数字]%。

这是[来源]给出的[指标名称]。

换句话说，[通俗解释]。

而[行业/领域]，正是[重灾区/关键点]之一。
```

**示例：**
```markdown
> 23%。

这是工信部《2025年人工智能就业白皮书》给出的AI总体替代率。

换句话说，每4个岗位中，就有近1个正被AI技术重构。

而财务，正是重灾区之一。
```

---

### 1.2 现象描述型开头

**模板：**
```markdown
> [金句：一句话概括核心观点/制造冲突]。

[具体数据/事件]正在[产生什么影响]。本文从[维度1]、[维度2]、[维度3]三个方面，结合[背景]，分析[主题]的深层原因与[启示/预警信号]。
```

**示例：**
```markdown
> 上市22年首亏，预亏70-90亿元。本文从盈利能力、运营效率、偿债能力三个维度，结合公司治理与行业周期，分析华发股份暴雷的深层原因与预警信号。
```

---

### 1.3 故事/案例型开头

**模板：**
```markdown
[时间/地点]，[人物]做了一件[出乎意料的事情]。

[描述事件]，结果[产生后果]。

这个[事件/案例]揭示了一个更大的问题：[核心问题]。

[转折引入主题]
```

---

## 2. 章节过渡模板

### 2.1 "但是"转折型

```markdown
但事情还有另一面。

同样的[调研/数据/分析]显示，**[对比数据]**。

为什么？因为有些[事物]，[做不到/无法实现]。
```

---

### 2.2 数据强调型

```markdown
**[核心结论]**。

这组数据揭示了一个触目惊心的趋势：**[数据解读]**。

更值得关注的是[指标名称]的变化。[详细说明]。
```

---

### 2.3 案例引出型

```markdown
先来看一个真实案例。

某[企业/组织/场景]，[引入变化]后：

- [变化1]：从[原始] → [改进]
- [变化2]：从[原始] → [改进]
- [变化3]：从[原始] → [改进]

**[效率/效果]提升了X倍**。
```

## 5. 分析段落模板

### 5.1 杜邦分析/财务分析型

```markdown
运用[分析方法]分解[指标]，可以更清晰地看到问题所在：

**[公式/关系]**。

[时间段]以来，[企业/现象]的[指标]变化主要源于[核心因素]。这说明公司的[核心能力]——[具体说明]——已经严重[受损/变化]。

[指标名称]是[行业]的生命线。[企业/现象]的[指标]持续[变化]，本质上反映了[深层原因]。
```

---

### 5.2 趋势分析型

```markdown

| [指标1] | [数值] | [标准] | [偏离] |
| [指标2] | [数值] | [标准] | [偏离] |

**[指标]意味着什么？**

从[专业视角]看，[指标定义]。[具体影响分析]。
```


---

### 6.2 判断型结论

```markdown

```

---

### 6.3 启示型结论

```markdown

```

## 9. 特殊段落模板


## 使用建议

1. **根据主题选择合适的模板**
2. **填充具体数据时注意准确性**
3. **保持整体风格的统一性**
4. **金句要简洁有力，15-25字为宜**
5. **表格数据要有来源标注**
6. **每节尽量有结论性段落**
