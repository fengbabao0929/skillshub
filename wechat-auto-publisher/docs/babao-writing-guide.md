## 文章结构分析

### 八宝说财审文章的三大类型

| 类型 | 特点 | 代表文章 |
|------|------|----------|
| **观点论证型** | 揭露伪命题、犀利观点、一针见血 | 《财务懂业务是个伪命题》 |
| **实务指南型** | 案例驱动、操作步骤、工具推荐 | 《预收款开票的税务处理》 |
| **深度分析型** | 数据支撑、多维度剖析、预警信号 | 《华发股份预亏90亿》 |

### 标准文章结构

```
标题（冲突式/数字冲击）
├── 导读（金句+价值概括）
├── 前言（场景引入/问题提出）
├── 正文（3-10节，根据内容灵活调整）
│   ├── 第一部分：问题/现象/背景
│   ├── 第二部分：分析/原因/论证
│   ├── 第三部分：核心观点/解决方案
│   ├── ...（根据需要增加）
│   └── 第N部分：行动建议/启示
├── 结语（总结+金句）
└── 相关阅读
```

**章节数量原则**：
- **实务指南**：4-7节（问题→原因→方案→步骤→注意事项→总结）
- **观点论证**：5-9节（现象→分析→案例→反驳→观点→建议→结语）
- **深度分析**：6-10节（概述→多维分析→预警→原因→对比→结论→启示）

## 标题公式大全

### 1. 冲突质疑型

**公式**：`[颠覆性观点]！[疑问/结论]`

**示例**：
- "财务懂业务是个伪命题！"
- "预收款开票是大忌！实务中到底该怎么处理？"
- "AI来了，财务人会被淘汰吗？"

### 2. 数字冲击型

**公式**：`[惊人数字]+[核心关键词]：[副标题]`

**示例**：
- "华发股份预亏90亿：基于财务视角的深度剖析"
- "23%的财务岗位正在被AI重构"
- "1441天存货周转：国企地产扩张模式破产"

### 3. 疑问痛点型

**公式**：`为什么[痛点]？[暗示有答案]`

**示例**：
- "为什么小公司财务总是乱？"
- "为什么你的预算总是不准？"
- "为什么同样是财务，他月薪3万你只有8千？"

### 4. 身份必看型

**公式**：`[目标人群]必看：[场景]+[收益]`

**示例**：
- "财务新人必看：入职前3个月要做好的3件事"
- "中小企业老板必看：财务混乱的5个信号"
- "创业公司财务必看：现金流管理5大误区"

### 5. 时间紧迫型

**公式**：`[时间节点]前必须[行动]，[后果]`

**示例**：
- "年底前必须完成的3项财务工作"
- "再过1个月，这3个税务优惠就失效了"
- "本月15号前必须申报！逾期罚款金额吓死人"

## 开头金句模板

### 1. 观点型金句

```markdown
> **[犀利观点8-15字]**

> **有些话，总得有人说。**
> **不是谁听谁的，而是互相成就。**
> **财务不需要成为业务专家，财务只需要成为财务专家。**
```

### 2. 数据型金句

```markdown
> [数字]%。

这是[来源]给出的[指标]。

换句话说，[通俗解释]。
```

### 3. 场景型金句

```markdown
> [一句话场景描述]，[后果/疑问]。

客户一句话：开票就付钱，不开票就不付钱。你开还是不开？
```

## 段落写作技巧

### 1. "翻译一下"风格

**用途**：把专业术语变成大白话

```markdown
不要说"权责发生制"，说"发货了才算收入"。
不要说"资本性支出"，说"买能用好几年的东西"。
不要说"经营性现金流"，说"实实在在收进来的钱"。

**翻译，不是降智。是让沟通更高效。**
```

### 2. 对比列表

```markdown
| 过去 | 未来 |
|------|------|
| 记账、算账、报账 | 分析、决策、创造价值 |
| 事后核算 | 事前预测、事中控制 |
```

### 3. ✅/❌ 符号列表

```markdown
**正在消失的工作**：
- 纯粹的核算会计
- 重复性的报表编制

**正在升值的工作**：
- 现金流预测与资金管理
- 定价模型与盈利分析
```

### 4. 案例分析三段式

```markdown
### X.X 案例背景

[时间、对象、问题简述]

### X.X 分析过程

[分析方法、数据、发现]

### X.X 处理结果

[解决方案、效果]
```

## 排版规范

### 1. 层级标题

```markdown
# 主标题（文章标题，不加在正文中）
## 一级章节（01/一/第一部分）
### 二级章节（1.1/核心原则）
#### 三级章节（小点说明）
```

### 2. 表格规范

```markdown
| 指标 | 2020年 | 2021年 | 2022年 |
|:---|:---:|:---:|:---:|
| 收入 | 100 | 120 | **150** |
| 利润 | 20 | 25 | 30 |

**数据来源：** 公司年报
```

### 3. 引用规范

```markdown
> 专家观点金句
> 引用法条
> 数据来源说明
```

### 4. 代码块规范

```markdown
**会计分录**：
```
借：银行存款 1130万
    贷：合同负债 1000万
    贷：应交税费——待转销项税额 130万
```
```

### 5. 结尾规范

```markdown
---

*如果这篇文章对你有帮助，欢迎点赞、转发。*

*关注我们，陪你一起在[时代]实现[目标]。*

---

**相关阅读**

- [文章标题](链接)
- [文章标题](链接)
```

## 快速上手

### Step 1: 使用文章模板

```bash
cp templates/babao-article-template.md my-article.md
```

### Step 2: 填写frontmatter

```markdown
---
title: 财务人必看的5个Excel技巧
subtitle: 从加班到准点下班的蜕变指南
author: 八宝
summary: 5个Excel技巧，让你的效率提升10倍
cover: ./images/cover.png
tags: 财务, Excel, 效率
date: 2026-01-15
---
```

### Step 3: 按章节结构填充内容

1. **开头**：用金句制造悬念/冲突
2. **第一节**：描述问题/现象
3. **中间节**：分析+案例+数据
4. **最后节**：行动建议/启示
5. **结尾**：总结+金句+行动号召

### Step 4: 检查必备元素

- [ ] 是否有开篇金句？
- [ ] 是否有3个以上表格？
- [ ] 是否有✅/❌列表？
- [ ] 是否有真实案例？
- [ ] 是否有行动建议？
- [ ] 是否有相关阅读？

### Step 5: 发布

```bash
node scripts/wechat-publisher.cjs --article my-article.md --action draft
```
