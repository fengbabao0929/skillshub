# 阶段02：标题优化

## 目标
**必须生成5-10个爆款标题供用户选择**

---

## 爆款标题公式

| 公式 | 示例 |
|------|------|
| 数字+收益 | "5个技巧，帮你节省30%成本" |
| 疑问+痛点 | "为什么效率低？因为这3个误区" |
| 对比+反差 | "同样做XXX，为什么他比你轻松？" |
| 身份+场景 | "财务人必看：金税四期应对指南" |
| 时间+紧迫 | "3月1日前，必须完成的3件事" |
| 警示+避坑 | "注意！这5个操作正在害你" |

---

## 标题评分标准

- 15-25字（含标点）
- 包含具体数字
- 直击痛点或利益
- 避免标题党词汇（震惊、奇迹）

---

## 输出格式

```
📌 为您生成的爆款标题：

1. 标题1 [数字+收益] ★评分
2. 标题2 [疑问+痛点] ★评分
3. 标题3 [对比+反差] ★评分
...
10. 标题10 [身份+场景] ★评分

请选择一个标题（输入序号）：
```
