# 阶段01：选题定位

## 目标
明确文章主题和定位

---

## 执行步骤

### 1. 确认写作主题
- 主题是什么？
- 要解决什么问题？
- 目标受众是谁？

### 2. 明确文章定位
- 文章类型：干货教程 / 观点文章 / 故事案例 / 列表盘点
- 预期字数：2500-3000字
- 核心价值：读者看完能获得什么？

---

## 输出格式

完成本阶段后，确认以下信息：

```
【主题】：XXX
【类型】：干货教程
【受众】：财务人
【字数】：2500-3000字
【核心价值】：XXX
```
