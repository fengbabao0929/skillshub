# 阶段06：配图提示词

## 目标
自动生成配图提示词并保存，然后询问用户是否生成图片

---

## 配图类型

### 1. 封面图（1张）
- 尺寸：900x383
- 位置：文章开头
- 内容：标题+核心主题视觉化

### 2. 信息图（2-3张）
- 尺寸：900x383
- 位置：正文中
- 内容：关键点可视化

### 3. 引导图（1张）
- 尺寸：900x383
- 位置：文章结尾
- 内容：关注引导

---

## 自动执行流程

### 6.1 生成配图提示词
根据文章内容自动生成5张图片的提示词：
1. cover.png - 封面图
2. info1.png - 信息图1
3. info2.png - 信息图2
4. info3.png - 信息图3
5. cta.png - 引导图

保存文件：与文章同名，添加 `_配图提示词.md`

### 6.2 询问用户
```
🎨 配图提示词已生成（共5张）

是否需要调用API生成图片？

回复'是'或'/wx-generate-images'开始生成
```

---

## 生成图片命令
```bash
python scripts/generate_images.py --input outputs/prompts.jsonl --out outputs/images
```

---

## 完成标志
用户确认生成图片后，整个创作流程完成。
