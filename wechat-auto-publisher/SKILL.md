- "搜索财经热点"

- "写一篇公众号文章"
- "生成爆款标题"
- "创作财务文章"
- "给文章配图"
- "/wx-trend" - 热点全景报告
- "/wx-create" - 交互式创作

## 热点搜索功能

### 1. 热点全景报告 `/wx-trend`

一键获取完整的热点信息，包括：
- 📈 热门话题 TOP5
- 📰 最新资讯 TOP5
- 🎯 爆款规律分析
- 💡 推荐写作话题 TOP5

```
/wx-trend all 10          # 全部分类
/wx-trend tax 10          # 只看税务
/wx-trend finance 10      # 只看财务
/wx-trend accounting 10   # 只看会计
```

### 2. 搜索热门话题 `/wx-hot`

搜索财经领域热门话题，每个话题包含：
- 标题和热度
- 分类和标签
- 推荐标题建议
- 话题原因

### 3. 获取行业资讯 `/wx-news`

获取最新财务、税务、政策资讯：
- 政策文件
- 行业动态
- 监管变化
- 重要通知

### 4. 分析竞品爆款 `/wx-viral`

分析同类账号爆款文章，提取：
- 爆款标题规律
- 内容类型分布
- 热门关键词
- 写作建议
- 发布时间建议

### 5. 推荐写作话题 `/wx-recommend`

基于热点、政策、竞品分析智能推荐：
- 综合评分排序
- 推荐指数
- 详细创作指南

## 爆款创作规律

### 6种标题公式

| 公式类型 | 模板 | 示例 |
|---------|------|------|
| 数字+收益 | {n}个{关键词}，{核心收益} | "5个内控制度，让小公司财务不再混乱" |
| 疑问+痛点 | 为什么{痛点}？{解决方案} | "为什么财务总是乱？建这5个制度就够了" |
| 对比+反差 | {负面}vs{正面}，{方法} | "同样做财务，为什么他月薪3万你只有8千？" |
| 时间+紧迫 | {时间}，{行动}+{后果} | "年底前必须做的3项工作，不然麻烦大了" |
| 身份+场景 | {身份}必看：{场景}+{方案} | "财务人必看：提升效率的5个方法" |
| 警告+揭秘 | {警告}！{揭秘内容} | "注意！这3个操作会被税务局稽查" |

### 黄金3秒开头

前300字必须完成：
1. **场景共情** - 用具体场景让读者产生代入感
2. **痛点揭露** - 指出问题的严重性和普遍性
3. **解决方案预告** - 给出希望，承诺解决方案

### 三段式内容结构

- **悬念冲突** → 提出问题，制造悬念
- **痛点放大** → 分析问题的严重性
- **解决方案** → 给出具体可执行的方案
- **价值升华** → 总结价值，上升到方法论

### 结尾转化四要素

1. 价值总结 - 用1-2句话总结核心价值
2. 互动引导 - 引导用户评论、分享
3. 关注引导 - 引导用户关注公众号
4. 分层福利 - 提供不同层次的福利（资料、咨询等）

## 配图功能

### 配图类型

| 类型 | 说明 | 数量 |
|------|------|------|
| **封面图** | 文章封面、分享预览 | 1张 |
| **信息图** | 正文内容配图 | 2-3张 |
| **引导图** | 结尾行动号召 | 1张 |

### 配图风格

- **质感**: 奶油色纸张底 + 彩铅线稿 + 淡水彩上色
- **尺寸**: 900×383 像素（头条封面推荐）
- **核心内容区**: 中心383×383像素（防裁剪）
- **风格**: 暖色调、轻涂鸦装饰、趣味但干净

## 使用流程

### 完整创作流程

```
搜索热点 → 选择话题 → 生成标题 → 创作文章 → 生成配图 → 发布
```

### 方式一：基于热点创作（推荐）

**步骤1：获取热点报告**

```
/wx-trend all 10
```

**步骤2：选择话题并生成标题**

```
/wx-titles "年${2024}个税专项附加扣除确认" 8
```

**步骤3：生成完整文章**

```
/wx-write "个税专项附加扣除确认" "年${2024}个税专项附加扣除开始确认！这5个变化财务人必知"
```

**步骤4：生成配图提示词**

```
/wx-images ./个税-article.md
```

**步骤5：生成图片**

```
/wx-generate-images ./个税-article_prompts.jsonl
```

**步骤6：发布**

```
/wx-publish ./个税-article.md
```

### 方式二：交互式创作

```
/wx-create
```

系统会引导你完成：
1. 输入文章主题
2. 设定目标受众
3. 选择标题（系统生成多个爆款标题供选择）
4. 确认大纲
5. 生成完整文章
6. 保存到本地
7. 询问是否生成配图
8. 生成配图提示词
9. 调用AI生成图片

## 核心功能模块

### 1. 热点搜索模块

- **热门话题搜索** - 搜索财经领域热点
- **行业资讯获取** - 获取最新财务、税务政策
- **竞品文章分析** - 分析爆款文章规律
- **话题推荐** - 智能推荐写作话题
- **热点全景报告** - 一键获取完整热点信息

### 2. 标题优化器

基于6种爆款公式生成标题，每个标题附带评分。

### 3. 文章生成器

按照爆款结构生成完整文章（黄金3秒开头 + 三段式内容 + 结尾转化）。

### 4. 配图提示词生成器

基于文章内容自动生成：
- 封面图提示词（大标题居中/左侧+装饰/三栏卡片）
- 信息图提示词（要点列表/流程步骤/对比双卡）
- 引导图提示词（关注引导/互动提问/行动号召）

### 5. 自动上传图片

- 自动识别文章中的图片引用
- 上传到微信素材库
- 替换为微信CDN链接

### 6. Markdown转HTML

- 自动转换Markdown为公众号支持的HTML格式
- 保留基本格式：标题、加粗、列表、引用等

## 配置说明

### 微信公众号配置

```
/wx-config
```

按提示输入：
- AppID（公众号后台 → 设置与开发 → 基本配置）
- AppSecret（同上，只能查看一次，需妥善保存）

### 图片生成API配置

```bash
# 复制配置文件
cp scripts/image.env.example scripts/image.env

# 编辑配置，填入API密钥
```

访问 [智谱AI开放平台](https://open.bigmodel.cn/) 获取API密钥。

### IP白名单配置

在公众号后台「设置与开发」→「基本配置」→「IP白名单」中添加服务器IP地址。

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| **IP白名单** | 需在公众号后台配置服务器IP白名单 |
| **图片大小** | 单张图片不超过2MB |
| **调用频率** | API有频率限制，避免短时间内大量调用 |
| **安全存储** | AppSecret仅存储在本地`.env`文件中 |
| **文章质量** | 爆款文章需要结合实际情况调整，生成后需人工审核 |
| **配图内容** | 核心内容需在中心383×383像素范围，防裁剪 |
| **热点时效** | 热点话题有一定时效性，建议及时创作 |

## 常见问题

**Q: 热点数据多久更新一次？**
A: 热点数据基于内置模拟数据，实际使用时可接入真实API。

**Q: 可以只看某个分类的热点吗？**
A: 可以，使用 `/wx-trend tax 10` 只看税务分类。

**Q: 生成的内容质量如何？**
A: 系统基于爆款规律生成内容框架和结构，具体内容需要根据实际情况调整优化。

**Q: 可以自定义创作规则吗？**
A: 可以修改`config/creation-rules.json`文件来调整创作规则。

**Q: 支持哪些类型的文章？**
A: 适用于各类知识分享、教程指南、经验总结类文章。

**Q: 配图生成需要什么？**
A: 需要智谱AI的API密钥，访问 https://open.bigmodel.cn/ 获取。

**Q: 发布失败怎么办？**
A: 检查：
1. AppID和AppSecret是否正确
2. IP白名单是否配置
3. 网络连接是否正常

## 快速开始

```bash

npm install

# 2. 配置公众号
npm run config

# 3. 测试连接
npm run test
```
