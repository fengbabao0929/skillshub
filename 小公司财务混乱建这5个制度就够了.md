## 引言：一个真实的故事

上周，一位朋友找我喝咖啡。

"老张，我公司快倒闭了。"

我吓了一跳："你公司去年生意不是挺好的吗？"

"生意是好，但账乱成一团。"

他掏出手机给我看：

- 供应商A说他欠款8万，他记录只欠5万
- 仓库里少了3台电脑，没人知道去哪了
- 员工报销的发票五花八门，还有餐饮票开成"办公用品"
- 最可怕的是，上个月税务局来查账，补税+罚款交了12万

"我招了个会计，但她啥都要问我，我不问她也懒得管。"

我叹了口气："你这不叫有会计，叫有个录入员。"

## 财务混乱的根源

小公司财务混乱，通常有3个原因：

| 原因 | 表现 |
|------|------|
| **没有规矩** | 老板口头说了算，今天一个样明天一个样 |
| **一人多职** | 财务兼行政兼人事，啥都管啥都不精 |
| **不敢管人** | 财务想管制度，但老板带头破坏 |

**结果就是：**

> 账是记了，但记的是糊涂账；
> 人是招了，但招的是个传话筒；
> 钱是赚了，但不知道赚去哪了。

---

## 核心问题：缺的不是人，是制度

很多老板以为财务混乱=招个高手来解决。

**错了。**

高手也需要制度支撑，否则一样干不好。

小公司不需要大公司的复杂流程，但**必须有最基本的规矩**。

今天给你5个制度，建起来，财务混乱问题解决80%。

## 制度一：资金支付审批制度

### 📌 解决什么问题？

| 混乱表现 | 后果 |
|----------|------|
| 老板口头让付款，没留痕 | 出了事没人担责 |
| 员工随意报销，没人审 | 钱被乱花 |
| 付款找不到依据 | 重复支付 |

### ✅ 简易版模板

```
【资金支付审批制度】

1. 付款三要素（缺一不可）
   □ 有合同/订单
   □ 有发票/收据
   □ 有验收单/入库单

2. 审批权限
   金额≤5000元：财务审核 + 老板签字
   金额>5000元：财务审核 + 老板签字 + 复核

3. 审批流程
   经办人申请 → 部门负责人确认 → 财务审核 → 老板审批 → 出纳付款

4. 付款后
   出纳在凭证上标注"已付X月X日"
   附件完整，装订归档
```

### 💡 落地建议

- **做成一张表**：付款申请单，把上面要素列出来
- **老板带头守**：你自己先按流程走，员工才会遵守
- **特殊情况也要留痕**：微信/邮件审批也要截屏保存

## 制度二：发票报销规范

### 📌 解决什么问题？

| 混乱表现 | 后果 |
|----------|------|
| 发票抬头错税号错 | 抵扣不了，损失税金 |
| 餐饮票开成办公用品 | 虚列支出，税务风险 |
| 报销没标准，看心情 | 员工不满，成本失控 |

### ✅ 简易版模板

```
【发票报销规范】

1. 发票四要素（错一个退回）
   □ 抬头正确：公司全称+税号
   □ 业务真实：与业务相关
   □ 项目清晰：办公用品写明细，不能开"一批"
   □ 时间合理：不能跨期太久

2. 不予报销的情形
   □ 没有抬头的个人发票
   □ 项目模糊的笼统发票
   □ 与业务无关的消费
   □ 超过规定标准的部分

3. 报销标准（示例）
   交通：据实报销（网约车/出租车票）
   餐饮：客户宴请需说明事由+参与人
   差旅：不同城市有不同标准（按公司制度）

4. 审批流程
   填写报销单 → 粘贴发票 → 部门负责人确认 → 财务审核 → 老板审批 → 打款
```

### 💡 落地建议

- **做个报销单模板**：把要素列清楚，员工照着填
- **发票错了就退回**：不要嫌麻烦，退两次他们就记住了
- **标准要透明**：什么能报、报多少，写清楚贴墙上

## 制度三：合同管理流程

### 📌 解决什么问题？

| 混乱表现 | 后果 |
|----------|------|
| 合同散落在各部门 | 对账时找不到 |
| 付款不看合同 | 多付、早付、错付 |
| 合同到期不知道 | 自动续约吃亏 |

### ✅ 简易版模板

```
【合同管理流程】

1. 合同三件事
   □ 签订前：财务审核付款条款
   □ 签订后：财务留存一份+登记台账
   □ 履行中：按合同条款付款

2. 合同台账（必备字段）
   合同编号 | 对方单位 | 合同金额 | 付款方式 | 付款进度 | 签订日期 | 经办人 | 附件位置

3. 付款对照
   每次付款前，必须核对：
   - 合同约定的付款节点到了吗？
   - 对方开了对应金额的发票吗？
   - 验收单/进度确认有吗？

4. 合同归档
   所有合同统一编号、统一存放
   电子版备份，防止丢失
```

### 💡 落地建议

- **用Excel做个台账**：不用买软件，Excel够用
- **合同编号规则**：年份-类别-序号（如2024-HT-001）
- **付款时必看合同**：把合同条款写在付款单备注里

## 制度四：资产盘点制度

### 📌 解决什么问题？

| 混乱表现 | 后果 |
|----------|------|
| 账上一台电脑，实际没有 | 资产流失 |
| 买了东西不登记 | 重复采购 |
| 员工离职不交接 | 资产被带走 |

### ✅ 简易版模板

```
【资产盘点制度】

1. 资产登记标准
   单价≥2000元、使用年限≥1年的物品，必须登记
   包括：电脑、打印机、办公家具、机器设备等

2. 资产台账（必备字段）
   资产编号 | 名称 | 购买日期 | 金额 | 使用部门 | 保管人 | 状态

3. 领用与归还
   领用时填写《资产领用单》，签字确认
   离职时必须归还，由财务确认签字

4. 定期盘点
   每半年盘点一次
   盘点结果：账实相符/盘盈/盘亏
   盘亏要追责，盘盈要入账

5. 标签管理
   每个固定资产贴标签：资产编号+保管人
```

### 💡 落地建议

- **从小做起**：先把电脑、打印机管起来
- **贴标签很重要**：一眼看出谁在用
- **盘点别走过场**：实地看、实物点

## 制度五：财务对账机制

### 📌 解决什么问题？

| 混乱表现 | 后果 |
|----------|------|
| 从不跟客户对账 | 应收账款越积越多 |
| 从不跟供应商对账 | 多付钱不知道 |
| 内部不对账 | 账实不符累积 |

### ✅ 简易版模板

```
【财务对账机制】

1. 外部对账
   □ 客户对账：每季度发一次对账单，确认应收金额
   □ 供应商对账：每半年核对一次应付金额
   □ 银行对账：每月拿到银行回单后立即核对

2. 内部对账
   □ 现金日记账：每天下班前核对现金余额
   □ 银行存款：每月核对银行账面余额
   □ 库存商品：每月核对账面与实物

3. 对账结果处理
   □ 相符：归档保存
   □ 不符：查明原因、调整账务、追究责任

4. 对账单模板
   日期 | 摘要 | 我方记录 | 对方记录 | 差异 | 原因 | 处理
```

### 💡 落地建议

- **固定时间做**：比如每月最后一天、每季度末
- **对账单要对方确认**：微信/邮件/签字都行，要有留痕
- **差异马上处理**：越拖越难查

## 制度怎么落地？3步走

### 第1步：开个会，宣贯一下

把财务、相关员工叫一起，说明白：

> 以前混乱不是人的错，是没有规矩。
> 现在规矩定好了，咱们按规矩来。

### 第2步：先做容易的

| 优先级 | 制度 | 理由 |
|--------|------|------|
| ⭐⭐⭐ | 资金支付审批 | 立即见效，老板最有动力 |
| ⭐⭐⭐ | 发票报销规范 | 员工最常接触 |
| ⭐⭐ | 合同管理流程 | 避免大额损失 |
| ⭐⭐ | 资产盘点制度 | 半年做一次就行 |
| ⭐ | 财务对账机制 | 可以逐步建立 |

### 第3步：坚持一个月

- **第一个月**：严格执行，不要例外
- **第二个月**：查漏补缺，优化流程
- **第三个月**：习惯养成，成为常态
